Course Modules

1. Excel Basics and Data Management
  • Spreadsheet navigation, formatting, and formulas.
  • Basic data management: filtering, sorting, and basic functions (SUM, AVERAGE, COUNT).
  • Introduction to data validation and pivot tables.
2. Advanced Excel for Data Analysis
  • Advanced formulas and functions (IF, VLOOKUP, INDEX, MATCH).
  • Data analysis with Excel’s Analysis ToolPak.
  • Creating and using pivot tables and pivot charts for data summarization.
  • Introduction to Excel macros for automation.
3. Power BI Fundamentals
  • Overview of Power BI and connecting data sources.
  • Data transformation with Power Query.
  • Creating visualizations and understanding data modeling.
  • Building interactive dashboards.
4. Advanced Power BI & Data Visualization Techniques
  • DAX (Data Analysis Expressions) for calculations.
  • Advanced visualizations, using Power BI’s custom visuals.
  • Publishing reports and sharing dashboards.
  • Data security and governance within Power BI.
5. Data Management and Collaboration with SharePoint
  • Overview of SharePoint for data management and document storage.
  • Setting up and managing libraries, lists, and permissions.
  • Integrating SharePoint with Excel and Power BI.
6. Data Collection and Forms with Microsoft Forms
  • Creating surveys, quizzes, and forms for data collection.
  • Analyzing collected data in Excel and Power BI.
  • Integrating Microsoft Forms with Power Automate for workflows.
